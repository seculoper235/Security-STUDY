** form 기반 인증 + REST API가 적용되지 않는 이유? **

form 기반 인증은 하나의 완성된 애플리케이션에서만 동작한다.
그 이유는 form 기반 인증은 login 페이지에서 form으로 전송받은 아이디/비밀번호로 인증을 진행하는 방식이다.
서버에게 비밀번호가 그대로 전송되는 등 간편하지만 프론트와 서버간의 긴밀한 관계로 맺어져 있다.
따라서 내부의 어떤 API들도 공개될 수 없고, 프론트와 서버가 합혀친 하나의 완전한 애플리케이션(Self-Contained App)에서만 적용 가능하다.
클라이언트에게 제공되는 것은 오직 완성된 로그인 페이지와 로그아웃 페이지일 뿐이다.

하지만 REST API는 완성된 것을 목표로 하지 않는다. 누구나 사용할 수 있도록 핵심 기능(API)만을 제공한다.
이렇게 여러 파티들에게 API들을 제공한다는 점은 노출되지 말아야 할 form 기반 인증과 맞지 않다.
따라서 REST API는 form 기반 인증을 사용할 수 없다.
